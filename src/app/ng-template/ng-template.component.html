Ng Template Component

ng-template is an angular element for rendering HTML. it is never displayed directly. it cna be displayed using structural directive that we use all the time: ngif, ngfor and ngswitch.,
befor rendering html, angular replaces ng-template with a comment. ng-template can be used with structural directive, viewcontainerref etc.,
if you put some html inside of an ng-template tag, it not won't be on the screen but it won't be in the DOM either. angular will replace the ng-template tag and its contents with a comment. the key is that ng-template will only be displayed if used in partnership with a structural directive.,

ng-template એ HTML rendering કરવા માટેનું angular element છે. તે ક્યારેય directly display થતું નથી. structural directive કે જેનો અમે હંમેશા use કરીએ છીએ: ngif, ngfor અને ngswitch, તેનો use કરીને display કરી શકાય છે,
html render કરવા માટે, angular ng-template ને comment સાથે બદલે(replace) છે. ng-template નો ઉપયોગ(use) structural directive, viewcontainerref વગેરે સાથે કરી શકાય છે.,
જો તમે ng-template tag ની અંદર અમુક html નાખો(put) છો, તો તે સ્ક્રીન(screen) પર નહીં હોય પરંતુ તે DOM(document object model) માં પણ નહીં હોય. angular ng-templatetag અને તેના contents ને comment સાથે બદલશે. મુખ્ય બાબત એ છે કે ng-template માત્ર ત્યારે જ display થશે જો તેનો ઉપયોગ structural directive સાથે partnership માં કરવામાં આવે.,

a rendered ng-template doesn't itself get turned info a DOM element, only the contents are rendered to the DOM.,
ng-template is a virtual element and its contents are displayed only when needed(based on conditions).,
ng-template should be used along with structural directives like ngif, ngfor,ngswitch or custom structural directives.,
ng-template never meant to be used like other html elements. it's an internal implementation of angular's structural directives.,
when you use a structural directive in angular we will add a prefix asterisk(*) brfore the directive name, this asterisk is short hand notation for ng-template.,
whenever angular encounter with the astrisk(*) symbol, we are informing angular saying thet it is a structural directve and angular will convort directive attribute to ng-template element.,
ng-template is not exactly a true web element. when we compile our code, we will not see a ng-template tag in html DOM.,
nagular will evaluate the ng-template element to convort it into a comment section in html DOM.
we woll se below example one by one :-
(1). use ng-template in angular
(2). using ng-template with *ngIf
(3). using ng-template with *ngFor
(4). using ng-template with ngSwitch

render કરેલ ng-template પોતે DOM(document object model) element ની information મેળવતું નથી, ફક્ત contents જ DOM ને render કરવામાં આવે છે,
ng-template એ વર્ચ્યુઅલ(virtual) element છે અને તેની contents માત્ર ત્યારે જ display થાય છે જ્યારે જરૂર હોય (based on conditions).,
ng-template નો use ngIf, ngFor, ngSwitch અથવા custom structural directives જેવા structural directives સાથે થવો જોઈએ.,
ng-template નો અર્થ ક્યારેય અન્ય HTML element જેમ ઉપયોગ કરવાનો નથી. તે angular ના structural directives નું internal implementation છે.,
જ્યારે તમે structural directives નો angular માં use કરો છો ત્યારે અમે directive ની આગળ એક prefix ફૂદડી(asterisk,substar,multiplication)(*) ઉમેરીશું નામ, આ ફૂદડી એ ng-template માટે short hand સંકેત(notation) છે.,
જ્યારે પણ (asterisk,substar,multiplication)(*) symbol સાથે angular સામસામે(encounter) આવે છે, ત્યારે અમે angular ને જાણ(informing) કરીએ છીએ કે તે એ છે structural directive અને angular ng-template element માટે directive attribute ને convort કરશે.,
ng-template એ exactly web element નથી. જ્યારે આપણે આપણો code compile કરીએ છીએ, ત્યારે આપણને html DOM માં ng-template tag દેખાશે નહીં.,
angular ng-template element નું મૂલ્યાંકન(evaluation) કરીને તેને html DOM માં comment section માં રૂપાંતરિત(convort) કરશે.

(2). using ng-template with *ngIf
(3). using ng-template with *ngFor
(4). using ng-template with ngSwitch
